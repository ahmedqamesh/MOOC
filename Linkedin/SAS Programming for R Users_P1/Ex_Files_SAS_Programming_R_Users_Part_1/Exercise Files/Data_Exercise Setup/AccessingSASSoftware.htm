Accessing SAS Software



To complete the exercises in this course, you must have access to SAS Studio, the web browser-based interface featured in SAS University Edition. Although SAS University Edition is available via Amazon Web Services (AWS), we recommend that you use the download version if you are not an experienced AWS user.

The download version of SAS University Edition is a free vApp that runs in virtualization software such as VirtualBox or VMware. The Quick Start Guide has step-by-step instructions for downloading and installing everything you need to get up and running. Be sure to follow the instructions for creating a shared folder so you can set up the course files in the next topic.

For more information about SAS University Edition, see the FAQ.
