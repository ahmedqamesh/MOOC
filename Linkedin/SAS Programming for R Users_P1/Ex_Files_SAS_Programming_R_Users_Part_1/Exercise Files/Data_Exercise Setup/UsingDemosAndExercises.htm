Using the Demos and Exercises



To use the demos and exercises for this course, you must first set up the data by following the instructions in SettingUpData.htm: Setting Up Data for this Course. You only need to set this data up once for the entire series of four courses.

The sp4r folder you create during data setup contains data files and SAS program files that are used in exercises and demonstrations.

Each chapter has an exercise folder named nn_exercises, where nn is the chapter number. This folder contains two PDF files for each exercise in the chapter: the exercise itself, and the corresponding solution. All data files that you need to use for an exercise are referenced by name in the PDF file.

Each chapter also has a corresponding folder named nn_demos, that contains a PDF file for each demonstration. This PDF file includes the name of the SAS program file that contains the demo code, and lists the steps needed to complete the demo.
